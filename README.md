# Batch-Audio-Transcription-Tool
支持录音、单文件、多文件和目录批量上传，调用支持阿里云 Fun-ASR / Cloudflare Whisper Large V3 Turbo 转写音频，并用机器翻译输出中英文结果/A local web tool that supports recording, single-file upload, multi-file upload, and batch directory upload. It transcribes audio with Aliyun Fun-ASR or Cloudflare Whisper Large V3 Turbo, then uses machine translation to output Chinese or English results.
