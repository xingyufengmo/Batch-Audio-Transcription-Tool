# 使用说明

## 启动

1. 打开 `config.json`，填写 DashScope、OSS、机器翻译的 Key。
2. 双击 `start.bat`。
3. 浏览器会自动打开 `http://127.0.0.1:8000`。

## 注意

- 本程序会在本机启动一个本地服务，只监听 `127.0.0.1:8000`。
- 第一次启动可能会被 Windows 防火墙或杀毒软件询问，允许本机访问即可。
- `config.json` 里有密钥，不要随意转发。
- 如果端口 `8000` 被占用，请先关闭占用该端口的程序。

## 目录说明

- `AliyunASRTranslator.exe`：主程序。
- `start.bat`：一键启动入口。
- `config.json`：云服务配置。
- `使用说明.md`：本文档。
